AI CAREER COPILOT - FREE OFFLINE ANALYSIS VERSION

1. Open terminal in this folder.
2. Install packages:
   pip install -r requirements.txt
3. Start the app:
   python App.py
4. Open:
   http://127.0.0.1:5000

Notes:
- OpenAI API key and API billing are NOT required.
- By default the app uses local SQLite database: career_copilot.db
- Forgot Password email OTP needs EMAIL and APP_PASSWORD in .env.
